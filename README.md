# it.cybersnow.io

Personal IT services site — hands-on IT support and security for small businesses and individuals.

## Stack
- Plain HTML + CSS (no build step needed)
- Hosted on Cloudflare Pages
- Custom domain: `it.cybersnow.io`

## Local Development
Just open `index.html` in your browser — no server required.

For live reload while editing, you can use VS Code's Live Server extension.

## Deployment
Push to `main` → Cloudflare Pages auto-deploys.

## Structure
```
it-cybersnow/
├── index.html       # Main homepage
├── styles.css       # All styles
├── assets/          # Images, favicon (add as needed)
└── README.md
```
